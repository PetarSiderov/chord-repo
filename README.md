## ğŸ¹ How to Use on Korg EK-50 / Pa50

### Setting Up Styles
1. **Navigate to STYLE** on your EK-50
2. **Select the appropriate category**: Pop, Rock, Dance, Latin, World, or Traditional
3. **Choose the suggested style** from the repertoire (e.g., "16Beat Pop" for beat songs, "Balkan 7/8" for Macedonian songs)
4. **Set the tempo** using the TEMPO dial to match the BPM range indicated

### Using Variations (VAR1-VAR4)
- **VAR1**: Use for intro and basic accompaniment
- **VAR2**: Add during verses for subtle percussion/drums
- **VAR3**: Full arrangement for chorus sections
- **VAR4**: Maximum energy for solos and finales

### Activating STS Layers
1. Press **STS1-STS4 buttons** to activate/deactivate layers
2. Recommended layers per genre:
   - **Ballads**: Piano (STS1), Strings (STS2), EP (STS3)
   - **Oriental/ÄŒoÄek**: Clarinet (STS1), Trumpet (STS2), Strings (STS3), Percussion (STS4)
   - **Starogradski**: Accordion (STS1), Piano (STS2), Strings (STS3)
   - **Rock/Beat**: EP (STS1), Brass (STS2), Strings (STS3)

### Transposing for Different Voices
- **Higher voice (tenor/soprano)**: Press **TRANSPOSE** and add +2 to +4 semitones
- **Lower voice (bass)**: Subtract -2 to -3 semitones
- Most songs are set for average male baritone voice

## ğŸ”„ GitHub Actions Workflow

This repository uses GitHub Actions to automatically generate the Word document.

### How It Works
1. **On Pull Request**: 
   - Converts `Repertoar-Balkan-4h.md` to `Repertoar-Balkan-4h.docx`
   - Uploads the DOCX as an artifact (available for 30 days)
   
2. **On Push to Main**: 
   - Converts the Markdown to DOCX
   - Commits the generated `Repertoar-Balkan-4h.docx` back to the repository

### Regenerating the Word File Locally

If you want to generate the DOCX file on your own computer:

**Prerequisites**: Install [Pandoc](https://pandoc.org/installing.html)

**Command**:
```bash
pandoc Repertoar-Balkan-4h.md -o Repertoar-Balkan-4h.docx --from markdown --to docx --standalone
```

### Generating PDF (Optional)

If you have LibreOffice installed:
```bash
# First generate DOCX (as above)
# Then convert DOCX to PDF
libreoffice --headless --convert-to pdf Repertoar-Balkan-4h.docx
```

Or use Pandoc with LaTeX:
```bash
pandoc Repertoar-Balkan-4h.md -o Repertoar-Balkan-4h.pdf --from markdown --pdf-engine=xelatex
```
